# meta-tracker
AI project decision tracker — maps the journey of building apps with AI tools

## Getting Started

```bash
npm install
npm run dev
```

> **Note:** This project depends on `jynaxxapps-tokens` (a private GitHub package at `github:Jynax/jynaxxapps-tokens`). `npm install` will fail without read access to that repository. If you're an external contributor, the design tokens are used for color/typography variables — you can substitute your own or remove the import.
